TupidBot

TupidBot sorgt mit einer Reihe von spaßigen und praktischen Befehlen für Unterhaltung im Chat.

Commands

TupidBot ermöglicht es dir, Befehle einfach im Chat auszuführen. Um einen Befehl zu nutzen, gib einfach das Präfix + gefolgt vom Command-Namen ein. Zum Beispiel, um den Ping-Befehl zu verwenden, schreibe +ping.
