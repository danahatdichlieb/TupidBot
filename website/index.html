TupidBot

TupidBot adds fun and practicality to the chat with a range of entertaining commands.

Commands

TupidBot allows you to easily execute commands in the chat. To use a command, simply type the prefix "+" followed by the command. For example: "+ping"

Features

With a variety of Commands, it makes the chat more engaging. Great for any chat that wants more fun and excitement!

Add the bot

To add TupidBot to your channel, just type +add in TupidBot's chat.
