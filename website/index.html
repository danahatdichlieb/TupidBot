TupidBot

TupidBot sorgt mit einer Reihe von spaßigen und praktischen Befehlen für Unterhaltung im Chat.

Commands

TupidBot ermöglicht es dir, Befehle einfach im Chat auszuführen. Um einen Befehl zu nutzen, gib einfach das Präfix "+" gefolgt vom Befehl ein. Zum Beispiel: "+ping"

Features

Mit einer Vielzahl von Commands sorgt er für mehr Unterhaltung im Chat. Ideal für jeden Chat, der mehr Spaß und Dynamik braucht!

Bot hinzufügen

Um TupidBot zu deinem Channel hinzuzufügen, schreibe einfach +add im Chat von TupidBot.
